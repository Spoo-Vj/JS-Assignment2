Datatypes in JavaScript


Types
1.Primitive Type
2.Non-Primitive Type
In Primitive type - 5
1. Number - Represents numeric values. Ex: var =5;
2. String - Represents sequence of characters. Ex: var y = "hello";
3. Boolean - Represents boolean value either false or true.
4. Undefined - Represents undefined value.
5. Null- No value.
In Non-Primitive type - 3
1.Object - Represents instance through which we can access members. Ex: var = {firstName:"John", lastName:"Doe"};
2.Array - Represents group of similar values. Ex: var = [1,2,4];
3.RegEx - Represents regular expression.
